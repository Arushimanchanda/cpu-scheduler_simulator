# Intelligent CPU Scheduler Simulator

A Python-based CPU scheduling simulator built for CSE316 - Operating Systems.

## Features
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Round Robin (with custom time quantum)
- Priority Scheduling

## How to Run
1. Install Python 3.x
2. Install matplotlib: pip install matplotlib
3. Run: python gui.py

## Files
- scheduler.py - Scheduling algorithms logic
- gantt.py - Gantt chart visualization
- gui.py - Graphical user interface

## Tech Stack
- Python 3
- Tkinter (GUI)
- Matplotlib (Visualization)
